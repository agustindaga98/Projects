# Food Delivery Time Analysis 🛵

Statistical analysis of the factors influencing food delivery 
time using multiple linear regression.

## Overview
Based on a dataset of 883 delivery orders, this project models 
delivery time as a function of distance, weather conditions, 
traffic level, vehicle type, and courier experience.

## Tools
- R (lm, stargazer, ggplot2)

## Techniques
- Descriptive statistics
- Multiple linear regression (OLS)
- One-hot encoding
- Log-linear transformations
- Linear hypothesis testing
- Permutation test (reshuffling)

## Files
- `Food_Delivery_Times.R` — full analysis code
- `reports/report.pdf` — report with results

## Context
Academic project developed in a postgraduate program 
in Managemen, Analytics & AI.

# Análisis de Tiempos de Entrega de Comida 🛵

Análisis estadístico de los factores que influyen en el tiempo 
de entrega de pedidos de comida mediante regresión lineal múltiple.

## Descripción
A partir de un dataset de 883 órdenes, se modeló el tiempo de 
entrega en función de la distancia, condiciones climáticas, 
nivel de tráfico, tipo de vehículo y experiencia del mensajero.

## Herramientas
- R (lm, stargazer, ggplot2)

## Técnicas aplicadas
- Estadística descriptiva
- Regresión lineal múltiple (MCO)
- One-hot encoding
- Transformaciones logarítmicas
- Test de hipótesis lineales
- Prueba de permutación (reshuffling)

## Archivos
- `Food_Delivery_Times.R` — código completo
- `reports/report.pdf` — informe con resultados

## Contexto
Proyecto académico desarrollado en el marco de un posgrado 
en Management, Analytics & IA.
